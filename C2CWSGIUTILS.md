# Instructions for easier migration from `c2cwsgiutils`

## Environment variables

Environment variable correspondence:

- `C2C_BASE_PATH` => no equivalent
- `C2C_SECRET` => no equivalent (insecure)
- `C2C_JSON_PRETTY_PRINT` => no equivalent
- `C2C_JSON_SORT_KEYS` => no equivalent
- `C2C_AUTH_GITHUB_CLIENT_ID` => `C2C__AUTH__GITHUB__CLIENT_ID`
- `C2C_AUTH_GITHUB_CLIENT_SECRET` => `C2C__AUTH__GITHUB__CLIENT_SECRET`
- `C2C_AUTH_GITHUB_SCOPE` => `C2C__AUTH__GITHUB__SCOPE`
- `C2C_AUTH_GITHUB_REPOSITORY` => `C2C__AUTH__GITHUB__REPOSITORY`
- `C2C_AUTH_GITHUB_ACCESS_TYPE` => `C2C__AUTH__GITHUB__ACCESS_TYPE`
- `C2C_AUTH_GITHUB_SECRET` => `C2C__AUTH__JWT__SECRET`
- `C2C_AUTH_GITHUB_COOKIE` => `C2C__AUTH__JWT__COOKIE__NAME`
- `C2C_AUTH_GITHUB_AUTH_URL` => `C2C__AUTH__GITHUB__AUTHORIZE_URL`
- `C2C_AUTH_GITHUB_TOKEN_URL` => `C2C__AUTH__GITHUB__TOKEN_URL`
- `C2C_AUTH_GITHUB_USER_URL` => `C2C__AUTH__GITHUB__USER_URL`
- `C2C_AUTH_GITHUB_REPO_URL` => `C2C__AUTH__GITHUB__REPO_URL`
- `C2C_AUTH_GITHUB_PROXY_URL` => `C2C__AUTH__GITHUB__PROXY_URL`
- `C2C_USE_SESSION` => no equivalent (state is stored in cookie using `C2C__AUTH__GITHUB__STATE_COOKIE` and `C2C__AUTH__GITHUB__STATE_COOKIE_AGE`)
